Linux如何查看CPU个数?

https://m.php.cn/article/486956.html

总逻辑CPU数 = 物理CPU个数 * 每颗物理CPU的核数 * 超线程数

查询CPU个数: cat /proc/cpuinfo | grep "physical id" | sort | uniq | wc -l

查询核数: cat /proc/cpuinfo | grep "cpu cores" | uniq

查询逻辑CPU总数: cat /proc/cpuinfo | grep "processor" | wc -l

-- 举例:

查询CPU个数:
[root@tbm-oracle1 ~]# cat /proc/cpuinfo | grep "physical id" | sort | uniq | wc -l
32

查询核数: 
[root@tbm-oracle1 ~]# cat /proc/cpuinfo | grep "cpu cores" | uniq
cpu cores       : 2

查询逻辑CPU总数:
[root@tbm-oracle1 ~]# cat /proc/cpuinfo | grep "processor" | wc -l
64
